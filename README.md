Taskbook – Manager de Sarcini cu Reminder Prezentare Generală Taskbook este o aplicație dezvoltată în Python care permite gestionarea sarcinilor zilnice prin adăugare, editare, ștergere și setare de remindere. Proiectul are ca scop implementarea unui sistem simplu și eficient de organizare a activităților personale.
________________________________________
Obiectivele Proiectului
-	Implementarea unui sistem de gestionare a sarcinilor
-	Crearea unei interfețe de utilizare clare și intuitive
-	Gestionarea datelor utilizatorului
-	Implementarea unui mecanism de reminder pentru sarcini
________________________________________
Tehnologii Utilizate
-	Python
-	Structuri de date (liste, dicționare)
-	Funcții și modularizare
-	Gestionarea timpului și a evenimentelor (pentru reminder)
________________________________________
Funcționalități Implementate
-	Adăugare sarcini
-	Editare sarcini existente
-	Ștergere sarcini
-	Listare sarcini active
-	Setare reminder pentru fiecare sarcină
-	Gestionarea stării sarcinilor
________________________________________
Competențe Tehnice Dezvoltate
-	Programare în Python
-	Organizarea logicii aplicației
-	Gestionarea datelor și a fluxului de execuție
-	Implementarea unui sistem de notificare/reminder
-	Structurarea unui proiect software
________________________________________
Posibile Extensii
-	Interfață grafică (GUI)
-	Salvarea persistentă a datelor (fișier JSON / bază de date)
-	Notificări automate la nivel de sistem
-	Integrare cu calendar
